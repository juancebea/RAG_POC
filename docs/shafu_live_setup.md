# Sha-Fu Live Setup

Sha-Fu uses a modern direct live setup designed for consistency, portability, and in-ear monitoring.

## Core live system

Sha-Fu uses a Soundcraft Ui24 mixer for live routing and monitor mixes.

Sha-Fu uses a B.Beat Limited Edition 128GB to run backing tracks, click tracks, cues, and reference material.

The band uses in-ear monitoring for live performances.

## Guitar setup

Oni uses a Neural DSP Quad Cortex as the main guitar processor.

The Quad Cortex can send stereo guitar outputs to front of house when the show supports stereo routing.

The Quad Cortex is used for modern high-gain rhythm tones, lead tones, clean tones, and clean tones with light overdrive.

## Bass setup

Xze uses a compact direct bass setup.

The bass signal can be sent directly to front of house and to the monitor system.

## Monitoring

Each band member can receive a separate monitor mix from the Soundcraft Ui24.

The live system can include click, cues, reference guitar, backing tracks, guitar, and bass in the monitoring setup.

## Recording

The Soundcraft Ui24 can be used to record rehearsals and live shows when configured for multitrack recording.
